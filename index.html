Step 1：建立 GitHub 儲存庫

  1. 1

    打開 GitHub

    點此直接前往「New repository」頁面 →
  2. 2

    填寫以下資訊

    Repository name：my-vision-portfolio
    Description：陳浩宇機器視覺工程師個人網站（可選）
    Visibility：Public（公開）
    Add a README file：勾選
  3. 3

    點擊綠色按鈕

    Create repository

Step 2：上傳你的網站檔案

最簡單拖曳上傳法

  1. 01 在新建立的儲存庫頁面，點擊「Add file」→「Upload files」
  2. 02 打開你的電腦資料夾 my-vision-portfolio
  3. 03 把裡面的所有檔案（index.html、live-server-guide.html 等）全部拖進網頁視窗
  4. 04 下方 Commit message 填「Initial commit」
  5. 05 點綠色按鈕 Commit changes

上傳完成後
